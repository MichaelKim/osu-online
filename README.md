# osu!online

osu! for the web!

## Notes

### Approach Rate

Total preempt:

- AR <= 5: 1800 - AR \* 120
- AR > 5: 1200 - (AR - 5) \* 120

Fade-in:

- AR <= 5: 1200 - AR \* 80
- AR > 5: 800 - (AR - 5) \* 100

### Overall Difficulty

- 300: 79.5 - OD \* 6
- 100: 139.5 - OD \* 8
- 50: 199.5 - OD \* 10

### Circle Size

- Radius = Playfield Width / 16 \* (1.7 - CS \* 0.14)

### Playfield

Always 4:3, stretched to screen
